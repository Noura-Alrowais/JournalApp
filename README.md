Welcome to Journali! This SwiftUI-based application allows users to create, edit, and manage their journal entries with ease. You can bookmark entries, search through them, and organize them by date.

With Journali you can:
- Create and Edit Entries: Users can add new journal entries and edit existing ones.
- Bookmark Entries: Easily bookmark your favorite entries for quick access.
- Search Functionality: Find entries using a search bar that filters by title and content.
- List Sorting: Organize entries by date or filter them to show only bookmarked entries.
- Customizable Interface: A clean and modern design with customizable colors and styles.
